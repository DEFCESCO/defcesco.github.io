Security Policy

Responsible Disclosure

I appreciate security researchers who help keep the internet safe. If you discover a security vulnerability, please follow these guidelines:

Contact Information

  • Email: austin@defcesco.io
  • PGP Key: Download PGP Public Key
  • Security.txt: View security.txt

Guidelines

  • Please allow reasonable time for response and remediation before public disclosure
  • Include detailed steps to reproduce the vulnerability
  • Avoid accessing or modifying user data
  • Do not perform denial of service attacks or degrade service

Scope

This policy applies to:

  • defcesco.io
  • blog.defcesco.io
  • Any associated subdomains

Out of Scope

  • Third-party services and integrations
  • Social engineering attacks
  • Physical security testing
